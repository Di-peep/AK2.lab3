# AK2.lab3

List of valid keys:
+ -h, --help, getting help;
+ -v, --verbose, enable verbose mode of program;
+ -l, --my_list, list of values;
+ -b, --my_boolean, optional flag;
+ -f, --my_file, saving file;

### Commands usage
`./Lab3`

`./Lab3 -h`

`./Lab3 --help`

`./Lab3 --verbose=4`

`./Lab3 -v 1`

`./Lab3 --my_list=3,4,5,6`

`./Lab3 -l 1,2,3,4`

`./Lab3 --my_file="some_text"`

`./Lab3 -f "some_text"`

`./Lab3 --my_boolean=0`

`./Lab3 -b=1`
